# 🛒 LUXE. - FakeStore API Catalog

## 📌 Descripción
Aplicación de catálogo moderno desarrollada con React y Tailwind CSS que consume datos de la FakeStore API. Permite explorar productos por categorías, realizar búsquedas en tiempo real y visualizar detalles específicos de cada artículo con un diseño responsivo y minimalista.

## 🛠️ Instalación
1. Clona el repositorio: `git clone https://github.com/CarjoTv/Mini-Proyecto.git`
2. Instala las dependencias: `npm install`
3. Inicia el proyecto: `npm run dev`

## ⚙️ Características
- **Listado de productos**: Grid dinámico y responsivo.
- **Búsqueda en tiempo real**: Filtrado instantáneo por nombre.
- **Filtro por categorías**: Navegación segmentada mediante rutas dinámicas.
- **Página de detalle**: Información completa y valoraciones de cada producto.
- **Manejo de estados**: Implementación de pantallas de carga (Loading) y gestión de errores.
